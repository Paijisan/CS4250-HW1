    Avis de décès, Hommage, Anniversaire de décès, naissance…
    Publiez l'annonce des grands événements de votre Vie dans Le Carnet du Monde.

    Ce site vous permet de rédiger vous-même et en quelques instants votre annonce à publier dans le journal Le Monde.

    Vous pouvez publier tous les événements de votre choix : avis de décès, de naissance, mariage, fiançailles, PACS, soutenance de thèse, etc.

    Les étapes de la publication :
    1. Choisissez l’édition du journal Le Monde dans laquelle votre annonce sera publiée
    2. Choisissez l’événement à annoncer
    3. Rédigez votre annonce
    4. Effectuez votre règlement sur notre site (paiement sécurisé par carte bancaire)
    5. Recevez la confirmation de votre commande. Votre carte est débitée à ce moment
